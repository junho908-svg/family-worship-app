# 🙏 우리집 가정예배 앱

> **v0.9 BETA** · 2026.04.21 · 말씀의지혜 채널과 함께

가족이 매일 함께 모여 짧게라도 주님 앞에 서는 것 — 이 단순한 일을 자연스럽게 만들기 위한 앱입니다.
어린이는 친구 같은 마스코트(로고몽)와 함께 성경 이야기를 즐기고,
어른은 시간대에 맞는 찬양과 가이드로 부담 없이 가정예배를 인도할 수 있습니다.

🌐 **Live Demo**: [family-worship-app-pi.vercel.app](https://family-worship-app-pi.vercel.app)

---

## ✨ 주요 기능

### 🌅 시간대별 BGM 자동 추천 (15곡 · 6 카테고리)
- 새벽기도 / 아침 묵상 / 기도 찬양 / 말씀 묵상 / 저녁 예배 / 잠들기 전
- 미니 플레이어 + 풀 플레이어 + 슬립 타이머 (15·30·45·60분)
- 즐겨찾기로 "우리집 플레이리스트" 만들기

### 📺 말씀의지혜 재생목록 (6 시리즈 · 42편 영상)
- 매일 10분 말씀 산책 / 기도해도 응답 없을 때 / 번아웃 크리스천을 위한 말씀
- 고난 중의 기다림 / 하나님의 뜻 분별하는 법 / 크리스천 재정 성경 말씀
- 앱 내 임베드 재생, 어떤 영상이든 풀스크린·PIP 지원

### 🐑 로고몽 마스코트 (9곳에서 살아 움직임)
- 헤더 / 웰컴 스크린 / 어린이 탭 / 성경 이야기 / 예배 완료 / 격려 모달 / 빈 상태 등
- 6가지 애니메이션 (떠다니기, 통통, 흔들흔들, 빼꼼, 축하 댄스 등)

### 📖 어린이 성경 이야기 (8편 + 퀴즈)
- 다윗과 골리앗 / 노아의 방주 / 다니엘과 사자굴 / 요셉의 꿈
- 모세와 홍해 / 아기 예수님 / 선한 사마리아인 / 오병이어
- 동화책 톤 + 핵심 성경 구절 + 가족 토론 질문

### 🙏 가족 기도 노트
- 기도 제목 추가 → 응답받으면 "응답" 섹션으로 이동
- 하나님의 신실하심을 시각적으로 누적 기록

### ⭐ 게이미피케이션
- 연속 출석 (🔥) · 스티커 적립 (⭐) · 이번 주 격려 모달
- 예배 완료 +3 / 응답 기도 +2 / 이야기·퀴즈·미션 +1

### 💌 카톡 공유 + 5단계 격려 시스템
- 오늘의 말씀을 한 번에 가족·친구와 나눔
- 출석 일수에 따라 자동으로 다른 격려 메시지

## 🚀 실행 방법

### 로컬 개발

```bash
git clone https://github.com/junho908-svg/family-worship-app.git
cd family-worship-app
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 프로덕션 배포

GitHub에 push하면 [Vercel](https://vercel.com)이 자동 배포합니다.
